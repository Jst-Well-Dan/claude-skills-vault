## 三步开始使用

### 第 1 步：安装 Claude Code GUI（5 分钟）

#### 下载和安装

1. **访问官方网站**（或 GitHub Releases 页面）
   - 下载链接：`[待补充]`
   - 选择适合你操作系统的版本：
     - Windows：`claude-code-gui-windows.exe`
     - macOS：`claude-code-gui-mac.dmg`
     - Linux：`claude-code-gui-linux.AppImage`

2. **安装过程**（和普通软件一样）
   - Windows：双击 `.exe` 文件，点击"下一步"安装
   - macOS：拖动到"应用程序"文件夹
   - Linux：给文件添加执行权限，双击运行

3. **第一次启动**
   - 双击桌面图标（或从开始菜单打开）
   - 看到欢迎界面

#### 配置 API Key

**什么是 API Key？**
- 就像你的"会员卡号"，让你可以使用 Claude 服务
- 需要在 Anthropic 官网获取（有免费额度）

**如何获取？**
1. 访问 [console.anthropic.com](https://console.anthropic.com)
2. 注册/登录账号
3. 点击"API Keys" → "Create Key"
4. 复制生成的 Key（看起来像 `sk-ant-xxxxx`）

**如何配置？**
1. 在 Claude Code GUI 中点击"Settings"（设置）
2. 粘贴你的 API Key
3. 点击"Save"（保存）
4. 看到 ✅ "Connected"（已连接）

**费用说明**：
- 新用户有免费额度（可以试用）
- 付费方案：按使用量计费，大约 $0.01 - $0.05 / 次对话
- 对于个人用户，一个月用不了多少钱（通常 < ¥50）

### Q2：数据安全吗？我的文件会被上传到云端吗？

**A：非常安全！**

**Claude Code 的工作方式**：
1. **在本地运行**：所有操作都在你的电脑上完成
2. **文件不上传**：你的文件永远不会离开你的电脑
3. **API 调用**：只有必要的文本内容（如文章分析）会通过加密连接发送到 Claude API，文件本身不会上传

**类比**：
- 就像你在本地用 Word 写文档，Word 不会把你的文档上传到微软服务器
- Claude Code 也是这样：在本地处理文件，只有需要 AI 分析的文本才会通过加密通道发送

**适合处理敏感数据**：
- ✅ 公司财务报表
- ✅ 客户资料
- ✅ 商业机密文档

### Q3：我能创建自己的 Skill 吗？

**A：可以！而且不难！**

**方法**：使用 **Skill Creator**（技能创建器）

**流程**：
```
1. 告诉 Skill Creator 你想创建什么 Skill
   例如："我想创建一个 Skill，帮我每周自动整理工作日志"

2. Skill Creator 会引导你：
   Q1：这个 Skill 什么时候被激活？（触发条件）
   Q2：需要哪些输入信息？（用户提供什么）
   Q3：输出什么结果？（生成什么文件）
   Q4：有什么特殊规则？（格式要求、标准）

3. 自动生成 Skill 文件（SKILL.md）

4. 保存到 Skills 文件夹，立即可用
```

**难度**：
- 简单 Skill（如格式转换）：10 分钟
- 中等 Skill（如特定分析流程）：30 分钟
- 复杂 Skill（如多步骤工作流）：1-2 小时

**完全不需要编程！**
